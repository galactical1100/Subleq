# Subleq
A programming puzzle-game about the one instruction computer subleq. Created with AXIOMconnect for Last Call BBS by Zachtronics, but also available to play on the web.
Play it on the web at https://stellartux.github.io/netronics-connect-online/?https://raw.githubusercontent.com/galactical1100/Subleq/main/subleq.js
Or play the game inside your Steam copy of Last Call BBS. In-game: go to Drive 0, Nettronics connect, top left arrow, add new servers, open folder, and drag in subleq.js.
